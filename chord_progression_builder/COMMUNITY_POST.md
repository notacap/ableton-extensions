**🎹 Chord Progression Builder — a free Live 12 extension for building progressions fast**

I made a Live 12 extension that adds a **"Build Chord Progression…"** right-click action
to any MIDI track. It opens already set to your Set's key and scale, so everything it
suggests is in key from the start.

What it does:

- 🎲 **Surprise me** rolls a musical 4-chord progression instantly
- **30+ genre presets** — Pop, EDM/House, R&B/Neo-Soul, Lo-fi, Jazz, Trap, Cinematic…
- **8 modes** with diatonic + curated borrowed chords
- Add **9ths / 13ths / sus** colors, pick **voicings** (close/open/drop-2/drop-3/wide,
  rootless & shell), with optional **auto voice-leading**
- **Rhythm patterns** — pads, stabs, 8th pulse, arps, strum — plus humanize
- **Preview it in the dialog**, tempo-synced, with piano / EP / synth sounds
- Hit **Write to Live** and it drops a looped MIDI clip on the track

No theory knowledge needed — pick sounds you like, hear them, write them.

**Requires Ableton Live Suite 12.4.5+.** Install by dragging the `.ablx` into
Preferences → Extensions.


Happy to hear feedback and feature requests! 🙏
